# Terminal Chatbot
- Esse projeto é um agente de chatbot via terminal usando Node.js.
- As bibliotecas que estão sendo usadas nesse projeto são a `Langchain` e `Groq`.

---
## Instalação e configuração
- Primeiramente, é necessário que você tenha o [node.js](https://nodejs.org/en/download) e o [git](https://git-scm.com/downloads) instalados em sua máquina;
- Com o primeiro passo feito, clone o repositório usando `git clone`:
```bash
git clone https://github.com/Andr3Dav1d/terminal-chatbot.git && cd terminal-chatbot
```
- Após isso, instale os pacotes necessários usando:
```
npm install
```
- E depois disso, inicie o código usando `node .` ou `node index.js`. A configuração é feita diretamente pelo terminal.
